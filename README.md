# 🍼 EcoPlay: Sustainable Joy for Tiny Humans

A sleek, lightweight **static storefront** for eco-friendly baby toys. This project focuses on a clean UI/UX to highlight sustainable materials and safe play.

## 🛠 Tech Stack
*   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
*   **Backend:** Node.js with Express
*   **Server Port:** 3000

## 🚀 Getting Started

1.  **Clone the repo:**
    ```bash
    git clone https://github.com
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Launch the server:**
    ```bash
    node server.js
    ```
4.  **View the magic:**
    Open [http://localhost:3000](http://localhost:3000) in your browser.

## ✨ Features
*   **Responsive Product Grid:** Perfectly scaled for mobile parents on the go.
*   **Static Performance:** Lightning-fast load times using Express to serve static assets.
*   **Eco-Labels:** Integrated badges for organic wood, BPA-free plastics, and recycled packaging.
